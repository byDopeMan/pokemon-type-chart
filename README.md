# Pokémon Typen Counter

Schnell nachschlagen welche Typen ein Pokémon treffen — und wie hart.  
Pokémon suchen oder Typen manuell kombinieren, sofort Schwächen, Resistenzen und Immunitäten sehen.

🔗 **[bydopeman.github.io/pokemon-type-chart](https://bydopeman.github.io/pokemon-type-chart/)**

---

## Features

**Pokémon-Suche**
- Über 1000 Pokémon suchbar (deutsch & englisch)
- Autocomplete mit Sprite-Vorschau
- Zuletzt gesuchte Pokémon werden gespeichert (klick ins Suchfeld)

**Typ-Analyse**
- Defensiv: Was trifft diesen Typ besonders hart / gar nicht?
- Offensiv: Gegen welche Typen ist dieser Typ am stärksten?
- Dual-Typ-Kombinationen (z.B. Feuer + Flug) werden korrekt berechnet
- 4×, 2×, ½×, ¼× und 0× Multiplikatoren auf einen Blick

**Fähigkeiten**
- Fähigkeiten die die Typ-Effektivität beeinflussen werden automatisch erkannt
- Beispiele: Schwebe (Boden-Immunität), Feuerfänger, Blitzableiter, Speckschicht
- Per Klick aktivieren/deaktivieren — Ergebnis aktualisiert sich sofort

**Sonstiges**
- Fee-Typ (Gen 6+) per Toggle ein-/ausblendbar
- Offline verfügbar als Download (einzelne HTML-Datei, kein Internet nötig)

---

## Tastenkürzel

| Taste | Aktion |
|-------|--------|
| `/` | Suche fokussieren |
| `Esc` | Alles zurücksetzen (Pokémon + Typen) |
| `↑` `↓` | In der Autocomplete-Liste navigieren |
| `Enter` | Pokémon auswählen / Suche bestätigen |

## Offline nutzen

Unter **[/download](https://bydopeman.github.io/pokemon-type-chart/download/)** kann die komplette App als einzelne HTML-Datei heruntergeladen werden — funktioniert danach ohne Internet, einfach die Datei im Browser öffnen.
